# Jaipur Rust Port (jaipur-2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the Jaipur card game from the old Go project (`~/Development/brdg.me/game/jaipur/`) to a new Rust crate `rust/game/jaipur-2`, reproducing every Go test in Rust with equivalent assertions and adding spec-defined contract/edge coverage.

**Architecture:** The crate follows the established `lost-cities-1`/`zombie-dice-2` pattern: `lib.rs` holds the `Game` struct with internal methods and the `Gamer` trait impl; `command.rs` holds the `Command` enum and parser combinators; `render.rs` holds `PubState`/`PlayerState` types and `Renderer` impls. A `GameRng` field on `Game` seeds deterministically. All mutation methods return `Vec<Log>`. Domain types use serde-derive enums (no gob/int-iota).

**Tech Stack:** Rust edition 2024, serde, rand 0.10.2, brdgme_game (Gamer trait, command parsers, GameRng, GameError), brdgme_markup (Node/Renderer), brdgme_color, brdgme_cmd (HTTP/CLI/REPL/Fuzz entry points).

## Global Constraints

- Port as `jaipur-2`; crate path is `rust/game/jaipur-2`; binary names use `jaipur_2_` prefix.
- Preserve Go behavior by default. Suspected source defects must be raised with the user and approved before fixing.
- Leave all changes unstaged. Do NOT stage, commit, or push.
- Existing tic-tac-toe-2 unstaged work must remain untouched — all registration additions must sit alongside tic-tac-toe-2 entries, never overwrite them.
- Reproduce every Go Jaipur test in Rust with equivalent assertions. No comments/test names/groupings pointing back to Go. Add spec-defined contract/edge coverage.
- Module boundaries: `command.rs` (Command enum + parsers) and `render.rs` (PubState/PlayerState + Renderer); this matches all existing Rust ports. No tiny single-type modules.
- **Command module:** `command.rs` is the preferred consistency choice because every existing Rust game crate uses it. The implementer may keep commands in `lib.rs` only if extraction would not form a cohesive independently understandable domain or would not materially improve readability. All file lists, task descriptions, and interface summaries in this plan assume `command.rs` is extracted.
- No panics in runtime paths; all errors are `GameError`.
- Final verification: `cargo fmt --all -- --check` and `cargo clippy --workspace --exclude web --all-targets -- -D warnings`.

    fn start_round(&mut self) -> Result<Vec<Log>, GameError> {
        let mut logs: Vec<Log> = vec![Log::public(vec![
            N::text("It is the start of the round, starting market with "),
            N::Bold(vec![N::Fg(
                Good::Camel.color().into(),
                vec![N::text("3 camels")],
            )]),
        ])];
        self.deck = initial_deck();
        self.deck.shuffle(&mut self.rng);
        self.market = vec![Good::Camel, Good::Camel, Good::Camel];
        self.replenish_market(&mut logs);

        self.camels = [0, 0];
        self.bonus_tokens = [0, 0];
        self.good_tokens = [0, 0];
        self.hands = [vec![], vec![]];
        self.tokens = [vec![], vec![]];
        for p in 0..NUM_PLAYERS {
            let hand: Vec<Good> = self.deck.drain(..5).collect();
            logs.extend(self.receive_cards(p, hand));
            self.tokens[p] = vec![];
        }

        self.goods = HashMap::new();
        for good in Good::trade_goods() {
            self.goods.insert(*good, good.token_values().to_vec());
        }

        self.bonuses = HashMap::new();
        for size in bonus_sizes() {
            let mut pile = bonus_values(size).to_vec();
            pile.shuffle(&mut self.rng);
            self.bonuses.insert(size, pile);
        }

        Ok(logs)
    }

    fn replenish_market(&mut self, logs: &mut Vec<Log>) -> bool {
        let n = 5usize.saturating_sub(self.market.len());

            return true;

            self.end_round(logs);
            return false;

            N::Bold(vec![N::text(format!("{}", drawn.len()))]),

        }
        let mut sorted = cards.clone();
        sorted.sort_by_key(|g| (*g as u8, std::mem::discriminant(g)));
        logs.push(Log::private(
            vec![N::text("You drew "), /* placeholder */ N::Bold(vec![N::text(format!("{}", cards.len()))]), N::text(" cards")],

use brdgme_game::command::parser::*;

use crate::{Game, Good};

#[derive(Debug, PartialEq, Clone)]
pub enum Command {
    Take { take: Vec<Good>, give: Vec<Good> },
    Sell { good: Good, quantity: usize },
}

impl Game {
    pub fn command_parser(&self, player: usize) -> Option<Box<dyn Parser<T = Command> + '_>> {
        if self.is_finished() || self.current_player != player {

pub fn take_goods(
    &mut self,
    player: usize,
    take_goods: Vec<Good>,
    give_goods: Vec<Good>,
) -> Result<Vec<Log>, GameError> {
    if !self.can_take(player) {
        return Err(GameError::invalid_input("can't take at the moment"));
    }
    let num_take = take_goods.len();
    let num_give = give_goods.len();
    if num_take == 0 {
        return Err(GameError::invalid_input("you must specify a good to take"));
    }
    if num_take == 1 && num_give != 0 {
        return Err(GameError::invalid_input(
            "if you are taking a single good you can't put any back",
        ));
    }
    if num_take > 1 && num_take != num_give {
        return Err(GameError::invalid_input(
            "if you are taking more than one good you must trade for the same number of goods in your hand, eg take gold dia for lea lea",
        ));
    }
    if num_take > 1 && take_goods.contains(&Good::Camel) {
        return Err(GameError::invalid_input(
            "the only way to take camels is using \"take camel\" which will take all the camels from the market and replace them with cards drawn from the deck",
        ));
    }

    let mut hand_size_after = self.hands[player].len() + num_take;
    for &g in &give_goods {
        if g != Good::Camel {
            hand_size_after = hand_size_after.saturating_sub(1);
        }
    }
    if hand_size_after > HAND_SIZE {
        return Err(GameError::invalid_input("that would exceed your hand size of 7"));
    }

    // Check same-type trading
    let mut take_map: HashMap<Good, usize> = HashMap::new();
    for &g in &take_goods {
        *take_map.entry(g).or_insert(0) += 1;
    }
    for &g in &give_goods {
        if let Some(&count) = take_map.get(&g) {
            if count > 0 {
                return Err(GameError::invalid_input("you can't trade the same type of good"));
            }
        }
    }

    // Check market has what we want
    let mut market_map: HashMap<Good, usize> = HashMap::new();
    for &g in &self.market {
        *market_map.entry(g).or_insert(0) += 1;
    }
    for (&good, &n) in &take_map {
        let available = market_map.get(&good).copied().unwrap_or(0);
        if available < n {
            return Err(GameError::invalid_input(format!(
                "the market only has {} {}",
                available,
                good.name()
            )));
        }
    }

    // Check we have enough goods to trade out of hand
    let mut for_map: HashMap<Good, usize> = HashMap::new();
    for &g in &give_goods {
        *for_map.entry(g).or_insert(0) += 1;
    }
    let mut hand_map: HashMap<Good, usize> = HashMap::new();
    for &g in &self.hands[player] {
        *hand_map.entry(g).or_insert(0) += 1;
    }
    hand_map.insert(Good::Camel, self.camels[player] as usize);
    for (&good, &n) in &for_map {
        let available = hand_map.get(&good).copied().unwrap_or(0);
        if available < n {
            return Err(GameError::invalid_input(format!(
                "you only have {} {}",
                available,
                good.name()
            )));
        }
    }

    let mut logs = vec![Log::public(vec![
        N::Player(player),
        N::text(" took "),
        // placeholder: render comma list of take_goods in Task 5
        N::Bold(vec![N::text(format!("{} cards", num_take))]),
    ])];

    // Remove take_goods from market
    for (&good, &n) in &take_map {
        for _ in 0..n {
            if let Some(pos) = self.market.iter().position(|&g| g == good) {
                self.market.remove(pos);
            }
        }
    }

    // Remove give_goods from player's hand/camels
    for (&good, &n) in &for_map {
        for _ in 0..n {
            match good {
                Good::Camel => {
                    self.camels[player] = self.camels[player].saturating_sub(1);
                }
                _ => {
                    if let Some(pos) = self.hands[player].iter().position(|&g| g == good) {
                        self.hands[player].remove(pos);
                    }
                }
            }
        }
    }

    self.market.extend(give_goods);
    self.hands[player].extend(take_goods);

    let mut replenish_logs = vec![];
    if self.replenish_market(&mut replenish_logs) {
        self.next_player();
    }
    logs.extend(replenish_logs);
    Ok(logs)
}

pub fn sell(&mut self, player: usize, good: Good, quantity: usize) -> Result<Vec<Log>, GameError> {
    if !self.can_sell(player) {
        return Err(GameError::invalid_input("you can't sell at the moment"));
    }
    if good == Good::Camel {
        return Err(GameError::invalid_input("can't sell that good type"));
    }
    let min_sales = good.min_sale() as usize;
    if quantity < min_sales {
        return Err(GameError::invalid_input(format!(
            "the minimum amount you can sell of that good is {}",
            min_sales,
        )));
    }
    let in_hand = self.hands[player].iter().filter(|&&g| g == good).count();
    if quantity > in_hand {
        return Err(GameError::invalid_input(format!(
            "you only have {} of that good",
            in_hand
        )));
    }

    let goods_pile = self.goods.entry(good).or_default();
    let num_tokens = quantity.min(goods_pile.len());
    let points: u32 = goods_pile[..num_tokens].iter().sum();
    self.tokens[player].extend_from_slice(&goods_pile[..num_tokens]);
    goods_pile.drain(..num_tokens);
    self.good_tokens[player] += num_tokens as u32;

    let mut suffix = String::new();
    if let Some(bonuses) = self.bonuses.get_mut(&quantity) {
        if let Some(bonus) = bonuses.first().copied() {
            self.tokens[player].push(bonus);
            self.bonus_tokens[player] += 1;
            bonuses.remove(0);
            suffix = " and took a bonus token".to_string();
        }
    }

    let mut logs = vec![Log::public(vec![
        N::Player(player),
        N::text(" sold "),
        N::Bold(vec![N::Fg(
            good.color().into(),
            vec![N::text(format!("{} {}", quantity, if quantity == 1 { good.name() } else { good.plural() }))],
        )]),
        N::text(format!(
            " for {} {}{}",
            points,
            if points == 1 { "point" } else { "points" },
            suffix,
        )),
    ])]);

    // Remove sold cards from hand
    for _ in 0..quantity {
        if let Some(pos) = self.hands[player].iter().position(|&g| g == good) {
            self.hands[player].remove(pos);
        }
    }

    // Check for 3 goods emptied
    let emptied = Good::trade_goods()

        .filter(|g| self.goods.get(g).map_or(true, |v| v.is_empty()))
        .count();

use brdgme_game::Renderer;
use brdgme_markup::{Align as A, Node as N};

use crate::{Good, PlayerState, PubState};

fn render_good(good: Good) -> N {
    N::Fg(good.color().into(), vec![N::Bold(vec![N::text(good.name())])])
}

fn render_goods_list(goods: &[Good]) -> Vec<N> {
    let mut sorted = goods.to_vec();
    sorted.sort_by_key(|g| *g as u8);
    let mut nodes: Vec<N> = vec![];
    for (i, g) in sorted.iter().enumerate() {
        if i > 0 {
            nodes.push(N::text("  "));
        }
        nodes.push(render_good(*g));
    }
    nodes
}

fn camel_display(count: u32) -> &'static str {
    if count == 0 {
        "no"
    } else {
        "some"
    }
}

fn render_token_table(goods: &std::collections::HashMap<Good, Vec<u32>>) -> Vec<N> {
    let trade = Good::trade_goods();
    // Build max height
    let max_height = trade.iter().map(|g| goods.get(g).map_or(0, |v| v.len())).max().unwrap_or(0);
    let mut rows: Vec<Vec<N>> = vec![];

    // Header row: good names
    let mut header: Vec<N> = vec![];
    for (i, g) in trade.iter().enumerate() {
        if i == 3 {
            header.push(N::text("")); // spacer between rare and common
        }
        header.push(N::Fg(g.color().into(), vec![N::Bold(vec![N::text(g.name())])]));
    }
    rows.push(header);

    // Token rows (bottom-up like Go render)
    for row_i in (0..max_height).rev() {
        let mut row: Vec<N> = vec![];
        for (i, g) in trade.iter().enumerate() {
            if i == 3 {
                row.push(N::text(""));
            }
            let pile = goods.get(g);
            if let Some(values) = pile {
                if values.len() > row_i {
                    let val = values[values.len() - 1 - row_i];
                    row.push(N::Fg(g.color().into(), vec![N::Bold(vec![N::text(val.to_string())])]));
                } else {
                    row.push(N::text(""));
                }
            } else {
                row.push(N::text(""));
            }
        }
        rows.push(row);
    }
    vec![N::Table(
        rows.into_iter()
            .map(|row| row.into_iter().map(|n| (A::Center, vec![n])).collect())
            .collect(),
    )]
}

impl Renderer for PubState {
    fn render(&self) -> Vec<N> {
        let mut out: Vec<N> = vec![];

        let remaining_rounds = 3u8.saturating_sub(self.round_wins[0] + self.round_wins[1]);
        out.push(N::Bold(vec![N::text(format!(
            "There {} {} {} remaining.",
            if remaining_rounds == 1 { "is" } else { "are" },
            remaining_rounds,
            if remaining_rounds == 1 { "round" } else { "rounds" },
        ))]));

        let leader_text = if self.round_wins[0] > self.round_wins[1] {
            "Player 0 is in the lead."
        } else if self.round_wins[1] > self.round_wins[0] {
            "Player 1 is in the lead."
        } else {
            "Scores are level."
        };
        out.push(N::text(leader_text));
        out.push(N::text("\n"));

        out.push(N::Bold(vec![N::text("Sale prices")]));
        out.push(N::text("\n"));
        out.extend(render_token_table(&self.goods));

        out.push(N::text("\n"));
        out.push(N::Bold(vec![N::text("Bonuses for selling")]));
        let bonus_str = format!(
            "3: {} left  4: {} left  5 or more: {} left",
            self.bonuses.get(&3).copied().unwrap_or(0),
            self.bonuses.get(&4).copied().unwrap_or(0),
            self.bonuses.get(&5).copied().unwrap_or(0),
        );
        out.push(N::text(bonus_str));
        out.push(N::text("\n\n"));

        out.push(N::Bold(vec![N::text("Market")]));
        out.push(N::text("\n"));
        out.extend(render_goods_list(&self.market));
        out.push(N::text(format!("\n\n{} cards left in the deck", self.deck_len)));

        out
    }
}

impl Renderer for PlayerState {
    fn render(&self) -> Vec<N> {
        let mut out = self.public.render();

        out.push(N::text("\n\n"));
        out.push(N::Bold(vec![N::text("You have")]));
        out.push(N::text("\n"));
        out.extend(render_goods_list(&self.hand));
        out.push(N::text(format!(
            "\n{} {}",
            self.public.camels[self.player],
            if self.public.camels[self.player] == 1 {
                "camel"
            } else {
                "camels"
            }
        )));
        out.push(N::text(format!(
            "\n{} point tokens",
            self.public.token_counts[self.player]
        )));

        let opponent = (self.player + 1) % 2;
        out.push(N::text("\n\n"));
        out.push(N::Bold(vec![N::text("Your opponent has")]));
        out.push(N::text(format!(
            "\n{} goods",
            self.public.hand_sizes[opponent]
        )));
        out.push(N::text(format!(
            "\n{} camels",
            camel_display(self.public.camels[opponent])
        )));
        out.push(N::text(format!(
            "\n{} point tokens",
            self.public.token_counts[opponent]
        )));

        out
    }
}
